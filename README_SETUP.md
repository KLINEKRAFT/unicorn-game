Unicorn Run GitHub Setup

1. Upload every file in this folder to the root of your GitHub repo.
2. Keep the assets folder name exactly as-is.
3. In GitHub:
   - go to Settings
   - open Pages
   - set Source to Deploy from a branch
   - choose main branch and /(root)
   - save
4. After GitHub Pages finishes, your game URL should be:
   https://klinekraft.github.io/unicorn-game/

Included:
- index.html
- style.css
- game.js
- assets folder with backgrounds, tiles, buttons, food, and unicorn_run sprite

Current build notes:
- portrait 9:16 layout for iPhone
- simple Press Start placeholder screen
- left / right / jump touch controls
- parallax background
- food collectibles for score
- no UI art used yet
